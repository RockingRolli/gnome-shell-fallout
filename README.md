# Install Font

The Fallout font "Monofonto" can be downloaded from: https://www.searchfreefonts.com/font/monofonto-2.htm

Install the font to: **/usr/share/fonts/truetype/monofont**

On Ubuntu reload the fonts:

> sudo fc-cache -f -v
